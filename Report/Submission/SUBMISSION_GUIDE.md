# Submission 상세 가이드

이 폴더에 제출본을 제출합니다.
## 1) 제출 파일명

- 기본 파일명: Team1_report
- 예시 파일명:
	- Team1_report.docx
	- Team1_report.pdf
	- Team1_report.tex
	- Team1_report.hwp

## 2) 제출 가능 템플릿/형식

- 허용 형식: hwp / docx / pdf / tex
- hwp는 별도 양식이 없으므로 docx 양식을 참조해 작성

## 3) 제출 기간

- 최종 보고서: 2026-06-12(금) 23:59까지

## 4) 분량 및 작성 규칙

- 본문 분량은 최대 2페이지 이하
- 참고문헌(References)은 페이지 수 산정에서 제외
- 템플릿 내용을 준용하되, 필요 시 subsection 또는 추가 섹션 기입 가능
- 그림/시각화가 많아 공간이 필요한 경우에 한해 extra +1페이지(총 3페이지)까지 허용

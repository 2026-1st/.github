##  시작하기

팀 배정 이후 각 팀은 다음 절차를 따릅니다.

1. 팀 GitHub 저장소를 생성합니다.
2. 팀원 전원을 Collaborator로 초대합니다.
3. `README.md`에 프로젝트 개요, 실행 방법, 환경 정보를 작성합니다.
4. Git을 처음 쓰는 팀원은 [Git / GitHub 가이드](../docs/Git_가이드.md)를 먼저 읽어 주세요.
